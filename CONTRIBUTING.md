# Contributing

Thanks for your interest in MonitorPI. This is a small, practical Pi kiosk project — contributions that keep it simple are welcome.

## How to contribute

1. Open an issue to discuss larger changes, or send a pull request for small fixes.
2. Fork the repo, create a branch, and open a PR against `main`.
3. Test on a real Raspberry Pi when changing GPIO, display, or Chromium behavior.

## Guidelines

- Keep scope small; match existing Bash and Python style in the repo.
- Do not commit `config.env` or personal hostnames, MAC addresses, or URLs.
- Update the [README](README.md) if you change setup steps or configuration options.

## Pull requests

- Describe what you changed and how you tested it.
- No contributor license agreement is required; you retain copyright on your contributions and license them under the same [MIT License](LICENSE) as the project.
